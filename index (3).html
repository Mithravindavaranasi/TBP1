🌿 About VaayuDrishti

VaayuDrishti (वायुदृष्टि) — meaning "Air Vision" — is a comprehensive air quality monitoring and analytics platform built for both citizens and policymakers in India.

We integrate data from WAQI real-time monitors, a browser-native Random Forest ML classifier (ported from Python scikit-learn), and a distributed IoT sensor network to provide the most granular picture of air pollution available.

The pollution source identification runs entirely in your browser — no backend server required. The ML model was trained on CPCB / Kaggle India city-day air quality data using pollutant ratio signatures to identify dominant sources such as vehicle emissions, stubble burning, industrial activity, and more.
